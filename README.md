# Персональний сайт про ШІ, програмістів і навчання Варіант 2 

Односторінковий статичний сайт, де я як студент комп’ютерних наук, ділюся думками про:
- програмування;
- штучний інтелект і його вплив на розробників;
- навчання, Linux та автоматизацію рутини;
- корисні AI-інструменти для розробників і студентів.

Сайт зроблено як навчальний/портфоліо-проєкт і розгорнуто як звичайну статичну HTML-сторінку.

---

## Основний функціонал

- **Hero-блок “Про автора”**  
  Коротко: хто я, чим займаюсь, які технології люблю (C++/Qt, HTML/CSS/JS/PHP, Linux Mint, Docker, AI tools).

- **Секції з контентом**  
  - пояснення, як ШІ взаємодіє з програмістами та впливає на їхню роботу;
  - плюси/мінуси використання AI-асистентів;
  - невеликий таймлайн навчання та розвитку;
  - навички та стек технологій;
  - чеклісти/кроки для ефективного використання AI.

- **Блок з AI-інструментами**  
  Таблиця та картки з основними сервісами: для коду, тексту, навчання, автоматизації тощо.

- **Відео та джерела**  
  Вбудовані відео з YouTube + список статей/ресурсів про ШІ та програмістів.

- **Адаптивна верстка**  
  Сітки на CSS Grid та Flexbox, коректна робота на десктопі та мобільних пристроях.

---

## Технології

- **HTML5** — семантична розмітка.
- **CSS3** — власний кастомний дизайн без фреймворків.
- Без бекенду та збірників: звичайний статичний сайт.
